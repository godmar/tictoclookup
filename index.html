Overview

This Google app provides a JSON interface to ticTOCs Journal Tables of Contents service, based on the data set provided here.

Usage as widgets

See Annette's IUG 2009 talk for details on how to use MAJAX 2, tictoclookup, and Google Book classes, particularly in the III Millennium OPAC.

Usage as a web service

This service allows querying by ISSN. In addition, a Title may be provided to disambiguate in cases where multiple feeds are offered for the same ISSN.

Examples:

  • Basic ISSN search:
    http://tictoclookup.appspot.com/0028-0836 
    {"lastmod": "Wed Apr 29 05:42:36 2009", 
 "records": [
    {"title": "Nature AOP", "rssfeed": "http:\/\/www.nature.com\/nature\/journal\/vaop\/ncurrent\/rss.rdf"}, 
    {"title": "Nature", "rssfeed": "http:\/\/www.nature.com\/nature\/current_issue\/rss"}
 ], 
 "issn": "00280836"}
  • Use of title= attribute:
    http://tictoclookup.appspot.com/0028-0836?title=Nature 
    {"lastmod": "Wed Apr 29 05:42:36 2009", 
 "records": [
     {"title": "Nature", "rssfeed": "http:\/\/www.nature.com\/nature\/current_issue\/rss"}
 ], 
 "issn": "00280836"}
  • Use of jsoncallback attribute:
    http://tictoclookup.appspot.com/0028-0836?title=Nature&jsoncallback=process 
    process({"lastmod": "Wed Apr 29 05:42:36 2009", 
         "records": [
           {"title": "Nature", "rssfeed": "http:\/\/www.nature.com\/nature\/current_issue\/rss"}
         ], 
         "issn": "00280836"})
  • Example with unknown ISSN (no search results):
    http://tictoclookup.appspot.com/0028-083X 
    {"lastmod": "Wed Apr 29 05:42:36 2009", "records": [], "issn": "0028083X"}
